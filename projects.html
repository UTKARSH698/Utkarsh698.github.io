CloudPulse Live
Python · Kinesis · S3 · Athena · DynamoDB · Terraform · React · Cognito · Glue
+

CloudFlow Live
Python · AWS Step Functions · Lambda · DynamoDB · SQS · CDK · LocalStack · X-Ray
+
Production-grade serverless order processing system using the SAGA choreography pattern. Coordinates reservation, payment, and confirmation across isolated Lambda functions without two-phase commit. Achieved 1,100+ req/min at 50 concurrent threads with P99 latency < 120ms.
Design Pattern
SAGA choreography — each service publishes events and reacts to others. No central orchestrator. Compensating transactions handle rollback automatically.
Ops Stack
GitHub Actions pipeline for build, test, Docker build, and Kubernetes deploy. Automatic rollback on failed health checks.
Why it matters: SAGA is how production systems handle distributed transactions. Building it from scratch demonstrates understanding of eventual consistency, idempotency, and failure modes.

CSPM Tool
Python · Lambda · EventBridge · SNS · S3 · CloudWatch · Terraform · GitHub Actions
+
Automated security scanning system checking AWS environments against CIS Benchmark v1.5 (IAM, S3, EC2, CloudTrail) on an hourly EventBridge schedule. Critical findings trigger SNS alerts in < 5 seconds; safe violations are auto-corrected; all findings logged to S3 for audit trail.
What It Detects
IAM role chaining, overly permissive policies, public S3 buckets, missing MFA enforcement, cross-account trust misconfigurations.
Output
Risk-scored findings (Critical/High/Medium) with Markdown remediation playbooks. CI-compatible — can block deployments on critical findings.
Why it matters: Security misconfigurations are the #1 cause of cloud breaches. This tool addresses the gap between "infrastructure deployed" and "infrastructure secured."

AgriFuture India Research
TLA+ · BFS Model Checker (Python) · Formal Verification · arXiv
+
Agricultural marketplace research thesis elevated into a formal CS contribution. Applied TLA+-style modelling to verify correctness properties of the marketplace's trust and access revocation system.
Research Finding
TAR (Trust and Access Revocation) preservation requires atomic revocation — partial revocation creates exploitable windows. Proven via BFS state-space exhaustion.
Artefacts
Custom BFS model checker in Python, TLA+-style spec, LaTeX-formatted arXiv preprint through multiple review passes.
Why it matters: Most student projects are "I built X." This one generated a novel research finding with a formal proof — a differentiator for TU Darmstadt and the research community.

Wikipedia Smart Search
Python · RAG · Vector Search · Embeddings · LLMs
+
Intelligent QA system over Wikipedia using Retrieval-Augmented Generation. Converts natural language questions into semantically relevant answers by embedding documents, indexing them via vector search, and passing retrieved context to an LLM for grounded responses.
How It Works
Wikipedia articles chunked and embedded → stored in a vector index → user query embedded and matched → top-k chunks passed to LLM as context → grounded answer returned.
Why RAG
Eliminates hallucination by grounding responses in retrieved source documents. Scalable to any corpus — Wikipedia today, internal knowledge bases tomorrow.
Why it matters: RAG is the dominant pattern for production LLM applications. Building it from scratch — not via a framework wrapper — demonstrates genuine understanding of embeddings, retrieval, and context window management.
